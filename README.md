## Basic Quiz app with Reports

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

[Demo](http://sreejithr.in/quiz/)

### Usage

1. Clone the project with `git clone https://github.com/sreejithr/quiz.git`.

2. Go inside the directory and do, `npm start` to start app in development mode at [http://localhost:3000](http://localhost:3000).
